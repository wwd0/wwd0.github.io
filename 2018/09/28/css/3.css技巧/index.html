2.通配符选择器作用？

日常开发中，通配符选择最大作用是剔除 部分 html标签的默认样式。 比如 a标签自带的下划线,蓝色字体， ol li标签前面的实心圆。还有各标签默认的margin,padding等属性。
*{
margin:0;
padding:0;
list-style:none;
text-deoration:none;
color:#000
}
